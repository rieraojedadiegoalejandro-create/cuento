**funcionamiento** 

Un mago de fuego controla el fuego o lo puede crear? 

Hay dos formas de usar la magia: moldear su energía para hacerla tuya y usarla, moldear tu energía para imitar la de tu elemento, en este sentido un mago de sol puede moldear la energía del sol para usarla como un arma, así como puede moldear la energía propia para imitar la del sol y producir luz, un mago de naturaleza puede usar la energía del crecimiento de las plantas para sanar, o puede usar imitar la energía de las plantas con la suya propia y usarla para hacer crecer semillas
